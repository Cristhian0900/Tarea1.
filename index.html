Servidores de Internet

Conoce los servidores más comunes: DNS, WEB, FTP, DHCP y Correo Electrónico

Servidor DNS

¿Qué es? El Sistema de Nombres de Dominio (DNS) es como la agenda telefónica de Internet. Traduce nombres fáciles de recordar, como www.google.com, en direcciones IP que las computadoras entienden.

¿Para qué sirve? Permite que podamos navegar en Internet usando nombres en lugar de memorizar números largos.

Ejemplo: www.google.com → 142.250.72.36

Servidor WEB

¿Qué es? Es un programa que almacena y entrega páginas web a los navegadores cuando un usuario las solicita.

Función: Responder a las peticiones de los usuarios (HTTP/HTTPS) enviando sitios web completos con texto, imágenes y videos.

Ejemplo: Cuando visitas YouTube, el servidor web de Google te envía la página principal y sus contenidos.

Servidor FTP

¿Qué es? FTP significa File Transfer Protocol. Se usa para transferir archivos entre una computadora y un servidor en Internet.

¿Para qué sirve? Permite subir o bajar documentos, imágenes y programas de manera rápida. Los usuarios suelen necesitar usuario y contraseña.

Ejemplo: Un programador actualiza su página web subiendo nuevos archivos al servidor por FTP.

Servidor DHCP

¿Qué es? El servidor DHCP (Dynamic Host Configuration Protocol) asigna direcciones IP automáticamente a los dispositivos que se conectan a una red.

¿Para qué sirve? Evita que tengas que configurar la IP manualmente en tu computadora o celular.

Ejemplo: Cuando conectas tu móvil al Wi-Fi de casa, el router actúa como servidor DHCP y le da una IP.

Servidor de Correos

¿Qué es? Son sistemas que permiten enviar y recibir correos electrónicos en Internet.

Enviar vs Recibir:

  • SMTP → para enviar correos.
  • IMAP → para recibir y sincronizar correos en varios dispositivos.
  • POP3 → para descargar correos en un solo dispositivo.

Ejemplo: Gmail usa SMTP para enviar mensajes e IMAP para que tus correos estén sincronizados en todos tus dispositivos.
